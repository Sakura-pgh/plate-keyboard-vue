# plate-demo 车牌键盘使用 Demo

> 车牌键盘控件 <br>
  支持：普通车牌/新能源车牌；特殊车牌（无牌车、WJ、使、民航、学、警、港、奥、领等）<br>
  支持：默认车牌输入；任意删除修改某一位数；优先显示当前省份简称 <br>

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Run your tests
